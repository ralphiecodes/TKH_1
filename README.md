Threat hunting 
"Threat hunting is the proactive effort in cybersecurity where analysts actively search for suspicious or malicious activity that may have evaded existing security controls, rather than waiting for automated alerts to indicate a breach. This systematic search aims to uncover threats that traditional security measures like firewalls or intrusion detection systems may miss, enabling organizations to detect hidden adversaries and reduce the time attackers remain undetected in a network."
